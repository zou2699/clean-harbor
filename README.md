# clean-harbor



## Usage

```shell
git clone https://github.com/zou2699/clean-harbor.git
cd clean-harbor
go build .
```



```shell
Usage of clean-harbor:
  -h    help message
  -keepNum int
        每个repo保留的tag个数 (default 5)
  -password string
        harbor账号
  -projectName string
        projectName
  -url string
        harbor地址
  -user string
        harbor账号
```
